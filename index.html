Stage Zero:

How the internet works: There are four main elements

  • Users: write and control content
  • Computers: run browsers which understand and display HTML coding
  • HTTP: transfers data from one computer or server to another
  • Servers: large computers that store websites and data

HTML & HTTP:

HTML is the basic language of websites that make up the internet

HTTP is the transfering of HTML documents from one computer to another

Elements and Tags:

Elements are a group of tags and contents. For the most part all tags have
a starting and a closing tag which define some attribute of the content.

Inline & Block Attributes:

Two attributes that are worth noting are inline and block. Inline are just that,
a change that is inserted in the middle of the content. Blocks on the other
hand create an invisible box around the content you are working with.

Tag Index:

  • < em > = emphasis = italics text
  • < a > = Anchor = link
  • < imp > = image
  • < br > = line break (does not need a closing tag, known as < void >)
  • < UL > = indent
  • < LI > = bullet point
  • < b > = bold text
  • < div > = encapsulates elements
  • < p > = paragraph
